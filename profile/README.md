# Permute — Portable Media Converter for Video, Audio, Images

## About Permute

Permute provides a complete media conversion environment for users who need to change video, audio, and image formats. Permute combines drag‑drop simplicity, hardware acceleration, and preset profiles. Permute supports all popular video formats (MP4, MOV, MKV, AVI, WebM), audio formats (MP3, AAC, FLAC, WAV, M4A, OGG), and image formats (JPEG, PNG, TIFF, GIF, HEIC, WebP). Permute integrates with Downie for seamless post‑download conversion. Permute enables users to drag any file, select a preset, and convert in one click. Permute runs on desktop platforms with hardware acceleration on modern chips. Permute is trusted by video editors, podcasters, content creators, students, and everyday users worldwide.
